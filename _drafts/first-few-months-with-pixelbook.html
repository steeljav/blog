Background:

Long time linux user.

Using looking versions of pycharm, intelij, rstudio and and a few others. Everything works.

Good:

  • No fans - no worries about getting hot and not getting airflow from sitting on constricted surface.
  • battery
  • responsiveness
  • keyboard - the best keyboard I have typed on in memory. Perfect balance between too mushy and too hard. Prefer something more firm.
  • sound - reasonable for the size
  • size / weight
  • overall build quality
  • Run chromeos, android apps (some) and linux software side by side!!!!

Not So Good:

  • Copying between linux container (AKA penguin) clipboard to chrome OS clipboard is not possible. This is an issue for me because I create analytic graphics in an app like RStudio and want to annotate and paste them into emails. Can't do this.
  • No good way to annotate graphics. End up pasting into google slides and creating a pdf slide instead of just annotating the graphic directly.
  • Linux app fonts are really small. May be a way to adjust this, but don't know. Some apps open very small.
  • cloud print not working, wtf?
  • some apps not working. not google's fault, not really developer's fault. but got no response from them: Plume.
  • run out of scroll / select room when doing a region select. sure I'm probably missing something. but having to put another finger on the trackpad to continue the selection seems like a mistake in interface design.
  • multiple desktops not yet implemented. there's a workaround involving multiple user accounts, but not really what I'm looking for.
  • stability of the dev channel could be better. weird things get broken, like scrolling of bookmarks list in chrome.

Differences:

  • The trackpad is different. Don't double tap to indicating end of selection, just lift. Minor issue but there's no button.

Recommendations:

  • Add a double tap to hold to the track pad.
